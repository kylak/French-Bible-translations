## NOTES

### Common Encoding Issues
- "Hé" (Hébreux/Hebrews) often fails to match due to UTF-8 encoding
  Solution: Add both "Hé" and "He" variants

- Accented characters (É, è, à) may not match in some systems
  Solution: Add both accented and non-accented variants

### Book Order Differences
- Protestant order (used here): 66 books
- Catholic order: Includes deuterocanonical books (Tobit, Judith, etc.)
- Jewish order (Tanakh): Different arrangement (Torah, Nevi'im, Ketuvim)

### Book Number Gaps
None. Numbers run consecutively 01-66.
